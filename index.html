e-GP Forensic Audit Rulebook & Engine ম্যাট্রিক্স (যেকোনো রুল ফিল্টার করতে ক্লিক করুন)

১. তথ্য গোপন (Rule: ITT 5.14) 0

স্ট্যান্ডার্ড টেন্ডার ডকুমেন্টের ITT 5.14 ধারা অনুযায়ী প্রতিটি বিডারের 'Beneficial Owner' বা প্রকৃত মালিকানার বিবরণ দেওয়া বাধ্যতামূলক।

২. চুক্তি বিভাজন (PPA 2006) 0

উচ্চতর মন্ত্রণালয়ের অনুমোদন এড়াতে সুনির্দিষ্ট উদ্দেশ্যে মেগা প্রজেক্টের কাজকে কৃত্রিমভাবে একাধিক ছোট ছোট লটে ভাগ করার নামান্তর।

৩. CPV কারসাজি (CPV Staffing) 0

জালিয়াতি করার জন্য মূল কাজের সাথে সম্পর্কহীন গোপন কোড (CPV) জোর করে ঢুকিয়ে সাধারণ ঠিকাদারদের কাছ থেকে টেন্ডার লুকানোর কারসাজি।

৪. বৈষম্যমূলক শর্ত (Tailored Criteria) 0

টেন্ডার ডকুমেন্টে নির্দিষ্ট এরিয়া বা নির্দিষ্ট ডেসিমাল টার্নওভারের অদ্ভুত শর্ত দেওয়া, যা শুধু আগে থেকে ঠিক করা সিন্ডিকেটের সাথে মিলে যায়।

৫. মনোপলি ফানেলিং (Rule: ITT 50.6) 0

১ জন রেস্পন্সিভ বিডার থাকলে রি-টেন্ডার আবশ্যক। কিন্তু স্ক্রিনিংয়ের মাধ্যমে বাকিদের অযোগ্য করে একক দরদাতাকে কাজ পাইয়ে দেওয়া।

৬. অস্বাভাবিক দর (Price Outliers) 0

প্রজেক্টের অফিশিয়াল বাজেট এস্টিমেটের চেয়ে চরম কম বা বেশি প্রাইস ড্রপ করে বাজেট ম্যানিপুলেশন (ALB Guidelines violation)।

৭. প্রি-টেন্ডার মিটিং প্রহসন 0

টেন্ডার প্রকাশের অত্যন্ত অল্প সময়ের ব্যবধানে মিটিং বা ক্লোজিং ডেট ফিক্সড করা, যা বাইরের ঠিকাদারদের প্রস্তুতির সুযোগ কেড়ে নেয়।

৮. বিড-রিগিং কোড (Dropout Ratio) 0

বিপুল সংখ্যক ঠিকাদার শিডিউল কেনার পরেও জমা না দেওয়া, যা সিন্ডিকেটের ভয়ভীতি বা আর্থিক রফার (Bid Rigging) সরাসরি প্রমাণ।
